# Multi-Asset Wallets

Enterprise wallet infrastructure for fiat, crypto, stablecoins, and digital securities. Custodial and self-custody options with 50+ chain support.

## Overview

Lux Financial provides complete wallet infrastructure:

- **Custodial Wallets**: Fully managed wallet operations
- **MPC Self-Custody**: Threshold signing with your key shares
- **Multi-Chain**: 50+ chains including Ethereum, Polygon, Solana
- **HSM-Backed**: Hardware security module integration

## Quick Start

```typescript
import { Lux } from '@luxbank/sdk';

const lux = new Lux({ apiKey: 'sk_live_...' });

// Create a multi-chain wallet
const wallet = await lux.wallets.create({
  type: 'custodial',
  chains: ['polygon', 'ethereum', 'solana'],
  label: 'Treasury Operations',
});

console.log(wallet.addresses);
// {
//   polygon: "0x1234...",
//   ethereum: "0x1234...",
//   solana: "Abc123..."
// }
```



### Transfers

```typescript

  from: walletA.id,

  amount: '5000.00',
  currency: 'USDC',
  chain: 'polygon',

```

## MPC Self-Custody

For institutions requiring complete control, our MPC solution enables self-custody with threshold signing.

### Setup

```typescript
import { LuxMPC } from '@luxbank/mpc';

const mpc = new LuxMPC({
  threshold: 2,
  parties: 3,
  keyShareHolders: [
    { id: 'ops', type: 'internal' },
    { id: 'security', type: 'internal' },
    { id: 'backup', type: 'cold' },
  ],
});
```

### Key Generation

```typescript
// Generate distributed wallet
const wallet = await mpc.generateWallet({
  chain: 'polygon',
});

// Key shares distributed to each party
```

### Threshold Signing

```typescript
// Sign requires 2-of-3 parties
const tx = await mpc.sign({
  walletId: wallet.id,
  transaction: {
    to: '0x...',
    amount: '10000.00',
    currency: 'USDC',
  },
});
```

### MPC Features

- 2-of-3, 3-of-5, or custom threshold schemes
- Air-gapped cold storage support
- Social recovery with trusted parties
- Full audit trail and signing ceremonies
