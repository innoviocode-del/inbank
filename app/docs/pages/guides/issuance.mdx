# Digital Securities

Issue stablecoins, security tokens, and tokenized assets with Lux Financial's compliant issuance infrastructure.

## Overview

Lux Financial provides complete digital securities infrastructure:

- **Stablecoin Issuance**: Launch fiat-backed stablecoins with audited reserves
- **Security Tokens**: Compliant equity, debt, and fund tokenization
- **Tokenized Assets**: Real estate, commodities, and alternative investments
- **Multi-Chain Deployment**: Ethereum, Polygon, Arbitrum, Lux, and 50+ chains

## Quick Start

```typescript
import { LuxIssuance } from '@luxbank/issuance';

const issuance = new LuxIssuance({
  token: 'ACMEUSD',
  chains: ['polygon', 'ethereum'],
});

// Check reserve status
const reserves = await issuance.getReserves();
console.log(reserves);
// {
//   totalSupply: "50000000.00",
//   reserves: {
//     cash: "25000000.00",
//     tbills: "25000000.00"
//   },
//   ratio: "1.00"
// }
```

## Stablecoin Issuance

### Reserve Management

```typescript
// Check reserves
const reserves = await issuance.getReserves();

// Mint new tokens (requires deposit)
await issuance.mint({
  amount: '1000000.00',
  chain: 'polygon',
  recipient: '0x...',
});

// Burn tokens (triggers withdrawal)
await issuance.burn({
  amount: '500000.00',
  chain: 'polygon',
});
```

## Smart Contracts

### Battle-Tested Templates

Built on OpenZeppelin with security features:

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

import "@openzeppelin/contracts/token/ERC20/ERC20.sol";
import "@luxbank/contracts/Mintable.sol";

contract AcmeUSD is ERC20, Mintable {
    constructor() ERC20("Acme USD", "ACMEUSD") {
        _setupRole(DEFAULT_ADMIN_ROLE, msg.sender);
    }

    function mint(address to, uint256 amount)
        external onlyMinter
    {
        _mint(to, amount);
    }

    function burn(uint256 amount) external {
        _burn(msg.sender, amount);
    }
}
```
