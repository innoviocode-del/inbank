# Global Payments

Send fiat and crypto payments to 200+ countries with real-time settlement. All currencies supported with local payment rails.

## Overview

Lux Financial provides complete global payment infrastructure:

- **200+ Countries**: Send payments to every country
- **180+ Currencies**: All major fiat and crypto currencies
- **50+ Payment Rails**: SWIFT, SEPA, ACH, SPEI, PIX, UPI, and local networks
- **Real-Time Settlement**: Instant cross-border payments

## Quick Start

```typescript
import { Lux } from '@luxbank/sdk';

const lux = new Lux({ apiKey: 'sk_live_...' });

// Send cross-border payment
const payment = await lux.payments.create({
  amount: '5000.00',
  sourceCurrency: 'USD',
  sourceMethod: 'stablecoin:USDC',

  destination: {
    country: 'MX',
    currency: 'MXN',
    method: 'SPEI',
    accountNumber: '014...',
    recipientName: 'Maria Garcia',
  },
});

console.log(payment);
// {
//   id: "pmt_abc123",
//   status: "completed",
//   fx_rate: "17.45",
//   amount_delivered: "87250.00 MXN",
//   fee: "5.00 USD"
// }
```

## FX Conversion

### Get Quote

```typescript
const quote = await lux.fx.getQuote({
  from: 'USD',
  to: 'EUR',
  amount: '10000.00',
});

console.log(quote);
// {
//   rate: "0.92",
//   amount: "9200.00 EUR",
//   fee: "25.00 USD",
//   validUntil: "2024-01-15T10:30:00Z"
// }
```

### Lock Rate

```typescript
// Lock rate for 24 hours
const lockedRate = await lux.fx.lockRate({
  quote: quote.id,
  duration: '24h',
});
```

### Execute Conversion

```typescript
const conversion = await lux.fx.convert({
  from: 'USD',
  to: 'EUR',
  amount: '10000.00',
  lockedRate: lockedRate.id,
});
```

## Compliance

### Automatic Screening

Every payment is automatically screened:

```typescript
const payment = await lux.payments.create({
  amount: '10000.00',
  destination: { ... },

  // Compliance is built-in
  compliance: {
    sanctions: true,  // OFAC, EU, UN
    pep: true,        // PEP screening
    aml: true,        // AML checks
  },
});

// Check compliance status
if (payment.compliance.status === 'cleared') {
  console.log('Payment approved');
} else {
  console.log('Manual review required');
}
```

## Batch Payments

### Create Batch

```typescript
const batch = await lux.payments.createBatch({
  payments: [
    {
      amount: '1000.00',
      destination: { country: 'MX', ... },
    },
    {
      amount: '2000.00',
      destination: { country: 'BR', ... },
    },
    // Up to 1000 payments per batch
  ],
});
```

### Track Batch

```typescript
const status = await lux.payments.getBatchStatus(batch.id);
// {
//   total: 100,
//   completed: 85,
//   pending: 10,
//   failed: 5
// }
```
