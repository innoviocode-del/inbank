# Third-Party Integrations

Lux Financial provides a configurable integration layer that allows you to connect your preferred providers for KYC, AML, stablecoins, banking, and more. Each integration can be enabled per-environment and configured via the API or dashboard.

## Integration Architecture

```
┌─────────────────────────────────────────────────────────────────┐
│                      Lux Financial API                          │
├─────────────────────────────────────────────────────────────────┤
│                    Integration Layer                            │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐          │
│  │   KYC    │ │   AML    │ │ Banking  │ │  Crypto  │          │
│  │ Adapter  │ │ Adapter  │ │ Adapter  │ │ Adapter  │          │
│  └────┬─────┘ └────┬─────┘ └────┬─────┘ └────┬─────┘          │
└───────┼────────────┼────────────┼────────────┼──────────────────┘
```

### Available Providers

| Provider | Type | Features | Regions |
|----------|------|----------|---------|
| **Jumio** | KYC/KYB | Document verification, biometrics, AML screening | Global |
| **Onfido** | KYC | AI-powered document & biometric verification | Global |
| **Persona** | KYC/KYB | Customizable verification flows | US, EU, Global |
| **Veriff** | KYC | Video-based identity verification | Global |
| **Sumsub** | KYC/KYB | Full compliance suite with transaction monitoring | Global |
| **Trulioo** | KYC/KYB | Global identity verification network | 195+ countries |
| **Alloy** | KYC/KYB | Orchestration layer for multiple data sources | US focus |
| **Socure** | KYC | AI/ML identity verification | US |
| **Plaid Identity** | KYC | Bank-verified identity | US |
| **Stripe Identity** | KYC | Document verification with Stripe | Global |

### Configuration

```typescript
// Configure KYC provider via API
await lux.integrations.configure({
  type: 'kyc',
  provider: 'jumio',
  credentials: {
    apiKey: process.env.JUMIO_API_KEY,
    apiSecret: process.env.JUMIO_API_SECRET
  },
  settings: {
    workflow: 'netverify',
    enableBiometrics: true,
    documentTypes: ['passport', 'driving_license', 'id_card'],
    callbackUrl: 'https://api.yourbank.com/webhooks/kyc'
  }
});
```

### Usage

```typescript
// Initiate KYC verification (uses configured provider)
const verification = await lux.kyc.initiate({
  customerId: 'cus_123',
  type: 'individual',
  redirectUrl: 'https://yourbank.com/kyc/complete'
});

// Returns provider-agnostic verification URL
// { verificationUrl: 'https://...', verificationId: 'ver_abc' }
```

| **Dow Jones Risk** | Risk & compliance data | News, PEP, sanctions |
| **LexisNexis** | Identity verification, risk | Public records |

### Configuration

```typescript
// Configure crypto AML provider
await lux.integrations.configure({
  type: 'aml',
  provider: 'chainalysis',
  credentials: {
    apiKey: process.env.CHAINALYSIS_API_KEY
  },
  settings: {
    riskThreshold: 'high', // low, medium, high
    enableRealTimeAlerts: true,
    monitoredAssets: ['BTC', 'ETH', 'USDC', 'USDT']
  }
});

// Configure traditional AML provider
await lux.integrations.configure({
  type: 'aml',
  provider: 'complyadvantage',
  credentials: {
    apiKey: process.env.COMPLYADVANTAGE_API_KEY
  },
  settings: {
    fuzziness: 0.6,
    searchTypes: ['sanctions', 'pep', 'adverse-media'],
    enableOngoingMonitoring: true
  }
});
```

### Usage

```typescript
// Screen crypto address
const addressRisk = await lux.aml.screenAddress({
  address: '0x1234...',
  chain: 'ethereum',
  provider: 'chainalysis' // optional, uses configured default
});

// { riskScore: 15, riskLevel: 'low', categories: [], alerts: [] }

// Screen entity with multiple providers
const screening = await lux.aml.screen({
  name: 'John Smith',
  dateOfBirth: '1970-01-15',
  providers: ['builtin', 'complyadvantage'] // fan out to multiple
});
```

### Configuration

```typescript
// Configure Bridge for stablecoin on/off-ramps
await lux.integrations.configure({
  type: 'stablecoin',
  provider: 'bridge',
  credentials: {
    apiKey: process.env.BRIDGE_API_KEY,
    apiSecret: process.env.BRIDGE_API_SECRET
  },
  settings: {
    defaultCurrency: 'USDC',
    supportedChains: ['ethereum', 'polygon', 'base'],
    webhookUrl: 'https://api.yourbank.com/webhooks/bridge'
  }
});

// Configure Circle for USDC operations
await lux.integrations.configure({
  type: 'stablecoin',
  provider: 'circle',
  credentials: {
    apiKey: process.env.CIRCLE_API_KEY
  },
  settings: {
    walletSet: 'production',
    idempotencyKeyPrefix: 'yourbank'
  }
});

// Configure custody provider
await lux.integrations.configure({
  type: 'custody',
  provider: 'fireblocks',
  credentials: {
    apiKey: process.env.FIREBLOCKS_API_KEY,
    privateKey: process.env.FIREBLOCKS_PRIVATE_KEY
  },
  settings: {
    vaultAccountId: 'your-vault-id',
    enableAutoSweep: true
  }
});
```

### Usage

```typescript
// Create stablecoin on-ramp (fiat -> USDC)
const onramp = await lux.stablecoin.onramp({
  customerId: 'cus_123',
  amount: 10000,
  sourceCurrency: 'USD',
  destinationCurrency: 'USDC',
  destinationChain: 'polygon',
  destinationAddress: '0x...',
  provider: 'bridge' // optional
});

// Create off-ramp (USDC -> fiat)
const offramp = await lux.stablecoin.offramp({
  customerId: 'cus_123',
  amount: 5000,
  sourceCurrency: 'USDC',
  sourceChain: 'ethereum',
  destinationCurrency: 'USD',
  destinationAccount: 'ext_acc_456'
});
```

### Configuration

```typescript
// Configure Plaid for account linking
await lux.integrations.configure({
  type: 'banking',
  provider: 'plaid',
  credentials: {
    clientId: process.env.PLAID_CLIENT_ID,
    secret: process.env.PLAID_SECRET
  },
  settings: {
    environment: 'production', // sandbox, development, production
    products: ['auth', 'transactions', 'identity'],
    countryCodes: ['US', 'CA'],
    webhookUrl: 'https://api.yourbank.com/webhooks/plaid'
  }
});

// Configure Modern Treasury for payments
await lux.integrations.configure({
  type: 'payments',
  provider: 'modern_treasury',
  credentials: {
    organizationId: process.env.MT_ORG_ID,
    apiKey: process.env.MT_API_KEY
  },
  settings: {
    defaultLedger: 'operating',
    enableReconciliation: true
  }
});

// Configure CurrencyCloud for FX
await lux.integrations.configure({
  type: 'fx',
  provider: 'currencycloud',
  credentials: {
    loginId: process.env.CC_LOGIN_ID,
    apiKey: process.env.CC_API_KEY
  },
  settings: {
    onBehalfOf: null, // or sub-account UUID
    defaultBeneficiaryCountry: 'US'
  }
});
```

### Usage

```typescript
// Link bank account via Plaid
const linkToken = await lux.banking.createLinkToken({
  customerId: 'cus_123',
  products: ['auth', 'transactions']
});

// Exchange public token after user completes Plaid Link
const linkedAccount = await lux.banking.exchangePublicToken({
  customerId: 'cus_123',
  publicToken: 'public-sandbox-xxx'
});

// Get account balance
const balance = await lux.banking.getBalance({
  accountId: linkedAccount.id
});

// Initiate ACH payment
const payment = await lux.payments.create({
  type: 'ach',
  amount: 1000,
  currency: 'USD',
  sourceAccount: 'int_acc_123',
  destinationAccount: linkedAccount.id,
  provider: 'modern_treasury' // optional
});
```

## MCP Server Integration

Lux provides a Model Context Protocol (MCP) server for AI-powered operations.

### Installation

```bash
npx @lux/mcp-server init
```

### Configuration

```json
{
  "mcpServers": {
    "lux-financial": {
      "command": "npx",
      "args": ["@lux/mcp-server"],
      "env": {
        "LUX_API_KEY": "sk_live_xxx"
      }
    }
  }
}
```

### Available Tools

| Tool | Description |
|------|-------------|
| `lux_get_customer` | Retrieve customer details |
| `lux_list_customers` | List customers with filters |
| `lux_create_customer` | Create new customer |
| `lux_screen_sanctions` | Screen against sanctions |
| `lux_get_wallet` | Get wallet details |
| `lux_list_wallets` | List wallets |
| `lux_create_transfer` | Create fund transfer |
| `lux_list_transfers` | List transfers |
| `lux_get_balance` | Get account/wallet balance |
| `lux_create_payment` | Initiate payment |
| `lux_get_rates` | Get FX/stablecoin rates |
| `lux_list_integrations` | List configured integrations |
