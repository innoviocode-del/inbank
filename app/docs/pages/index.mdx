## Platform Overview

Lux Financial offers a complete neobank infrastructure stack:

### Core Banking
- **Native Stablecoin Support**: USDC, USDT, PYUSD, and custom stablecoins
- **International Payments**: Real-time settlements to 150+ countries
- **Multi-Currency Support**: USD, EUR, GBP, and 35+ fiat currencies
- **Multi-Chain Deployment**: Polygon, Ethereum, Arbitrum, Optimism, Base

### Enterprise Infrastructure
- **Lux KMS**: Enterprise key management with HSM integration
- **Lux MPC**: Multi-party computation for self-hosted custody
- **Lux IAM**: Enterprise IdP with SSO and role-based access
- **Post-Quantum Security**: Future-proof cryptography via Lux Node

### AI-Powered Operations
- **MCP Server**: Model Context Protocol for AI-powered bank operations
- **ZAP Protocol**: Browser-to-MCP real-time communication
- **AI Compliance**: Automated KYC/KYB and transaction monitoring
- **AI Chat**: Customer support and admin operations via AI
