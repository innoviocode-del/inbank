# Customers

Customers represent individuals or businesses that use your platform. Each customer can have multiple accounts, wallets, and external accounts.

## The Customer Object

```json
{
  "id": "cus_abc123",
  "object": "customer",
  "type": "individual",
  "email": "john@example.com",
  "first_name": "John",
  "last_name": "Doe",
  "phone": "+1234567890",
  "status": "active",
  "kyc_status": "approved",
  "created_at": "2026-01-15T10:00:00.000Z",
  "updated_at": "2026-01-15T10:00:00.000Z"
}
```

### Attributes

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique identifier (cus_*) |
| `object` | string | Always "customer" |
| `type` | string | "individual" or "business" |
| `email` | string | Customer email address |
| `first_name` | string | First name (individuals) |
| `last_name` | string | Last name (individuals) |
| `business_name` | string | Company name (businesses) |
| `phone` | string | Phone number (E.164 format) |
| `status` | string | "pending", "active", "suspended" |
| `kyc_status` | string | "pending", "in_review", "approved", "rejected" |

---

## Create a Customer

Creates a new customer.

```
POST /v0/customers
```

### Request Body

```json
{
  "type": "individual",
  "email": "john@example.com",
  "first_name": "John",
  "last_name": "Doe",
  "phone": "+1234567890",
  "address": {
    "line1": "123 Main St",
    "city": "New York",
    "state": "NY",
    "postal_code": "10001",
    "country": "US"
  }
}
```

### Response

```json
{
  "id": "cus_abc123",
  "object": "customer",
  "type": "individual",
  "email": "john@example.com",
  "first_name": "John",
  "last_name": "Doe",
  "status": "pending",
  "kyc_status": "pending",
  "created_at": "2026-01-15T10:00:00.000Z"
}
```

### Example

```typescript
const customer = await lux.customers.create({
  type: 'individual',
  email: 'john@example.com',
  first_name: 'John',
  last_name: 'Doe',
  phone: '+1234567890',
  address: {
    line1: '123 Main St',
    city: 'New York',
    state: 'NY',
    postal_code: '10001',
    country: 'US',
  },
});
```

## List Customers

```
GET /v0/customers
```

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `limit` | integer | Number of results (1-100, default 20) |
| `cursor` | string | Pagination cursor |
| `status` | string | Filter by status |
| `kyc_status` | string | Filter by KYC status |

### Response

```json
{
  "object": "list",
  "data": [
    {
      "id": "cus_abc123",
      "object": "customer",
      ...
    }
  ],
  "has_more": true,
  "cursor": "cur_xyz789"
}
```

## Submit KYC Documents

Submit identity verification documents for a customer.

```
POST /v0/customers/{customer_id}/kyc
```

### Request Body

```json
{
  "document_type": "passport",
  "document_front": "data:image/png;base64,...",
  "document_back": "data:image/png;base64,...",
  "selfie": "data:image/png;base64,..."
}
```

### Response

```json
{
  "id": "kyc_xyz789",
  "object": "kyc_submission",
  "status": "in_review",
  "created_at": "2026-01-15T10:00:00.000Z"
}
```
