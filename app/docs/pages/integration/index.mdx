# Integration Guide

This guide covers integrating Lux Financial into your application.

## Integration Options

### 1. REST API

Direct API integration for full control:

```typescript
const response = await fetch('https://api.lux.financial/v1/payments', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    amount: 10000,
    currency: 'USD',
    // ...
  }),
})
```

### 2. SDK Integration

Use our official SDKs for faster development:

- [Node.js SDK](https://github.com/luxfi/bank-sdk-node)
- [Python SDK](https://github.com/luxfi/bank-sdk-python)
- [Go SDK](https://github.com/luxfi/bank-sdk-go)

### 3. White-Label Embed

Embed our pre-built UI components:

```html
<script src="https://js.lux.financial/v1/embed.js"></script>
<div id="lux-payment-form" data-api-key="pk_live_..."></div>
```

## Webhook Setup

### 1. Configure Endpoint

Register your webhook URL in the dashboard or via API:

```typescript
await client.webhooks.create({
  url: 'https://your-app.com/webhooks/lux',
  events: ['payment.completed', 'payment.failed'],
  secret: 'whsec_...',
})
```

### 2. Verify Signatures

```typescript
import { verifyWebhookSignature } from '@luxfi/bank-sdk'

app.post('/webhooks/lux', (req, res) => {
  const signature = req.headers['x-lux-signature']

  if (!verifyWebhookSignature(req.body, signature, webhookSecret)) {
    return res.status(401).send('Invalid signature')
  }

  // Process webhook
  const { event, data } = req.body

  switch (event) {
    case 'payment.completed':
      await handlePaymentCompleted(data)
      break
    case 'payment.failed':
      await handlePaymentFailed(data)
      break
  }

  res.status(200).send('OK')
})
```
